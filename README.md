# Snapmaker Tools

Utilities for Snapmaker workflows.

## Tools

### Snapmaker Orca Filament Importer

Windows helper for importing bundled Orca-style filament profiles into Snapmaker Orca as custom user filament presets for the Snapmaker U1.

Location:

```text
snapmaker-orca-filament-importer
```

Quick start:

1. Download or clone this repo.
2. Open `snapmaker-orca-filament-importer`.
3. Close Snapmaker Orca.
4. Double-click `Run-SnapmakerOrcaFilamentImporter.cmd`.
5. Choose source, brand, and filament series.

See [snapmaker-orca-filament-importer/README.md](snapmaker-orca-filament-importer/README.md) for details.
